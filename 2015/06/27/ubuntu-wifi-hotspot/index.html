Ubuntu 14.04 开启Wifi热点

发表于 2015-06-27 | 更新于 2016-04-16 | 分类于 Linux
不需要安装任何程序,修改配置即可开启WiFi热点.

前言

网上有好几种方式，但是要么不支持Android，要么需要安装第三方应用。本文介绍的是直接使用系统的网络管理创建Wifi，开启方便。
而且理论上应该是Ubuntu 14.04+都能支持，开启的热点各种终端都能用。
可惜不支持IPv6。

本文适用于Ubuntu新手，主要以图片展示过程。

修改网络模式

之前建立的网络，如果网络模式选择ad-hoc建立的热点，很多终端也是能用的，但是不支持Android,接下来就让它成为真正的热点，供给所有终端，包括Android的。

1. 在文件管理器里打开下面目录

/etc/NetworkManager/system-connections/
里面显示的是目前已经建立的所有网络连接

2.修改网络连接

找到刚才建立的连接名，我的是Wi-Fi Hotspot，修改该文件。

$ sudo gedit /etc/NetworkManager/system-connections/Wi-Fi\ Hotspot

最后一个名字就是之前建立的网络名字，在终端输入时，打名字首字母，然后按一下tap键即可自动补全。

将mode=infrastructure 改为mode=ap，保存即可

打开Wi-Fi热点

在网络连接的地方，选连接到隐藏的网络，找到刚建立的网络连接就ok。

总结

接下来就享受Ubuntu Wi-Fi带来的便利体验吧！
每次开启只需要执行最后一步，“打开Wi-Fi热点”就OK。
