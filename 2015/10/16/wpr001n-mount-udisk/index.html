品胜 WPR001N 路由器挂载U盘扩展磁盘空间

发表于 2015-10-16 | 更新于 2017-02-14 | 分类于 OpenWrt
扩展路由系统存储空间，WPR001N 和 WR703N 通用

前言

(2017.2更新)OpenWrt 15.05.1升级了内核，该方式不可用，查看新的设置方式：
https://ifhw.github.io/2017/02/14/wr703n-mount-udisk/

WR703N 存储空间是4M，WPR001N存储空间8M，RAM 都是64M
不管是4M还是8M,刷OpenWrt的固件后使用都得悠着点，不能随意的安装应用，毕竟刷完系统剩余空间要以KB计算
挂载U盘作存储之后，就可以任性的安装应用，存储等等了，想装什么装什么
目前(2015.10)我刷的OpenWrt 15.05 Chaos Calmer

准备工作

准备一个U盘，我选择的是8G的，刚好有闲置的，给U盘分区，5G ext4,剩余的 swap （分区工具推荐Linux下的GParted）

以下命令基本都是ssh登录到路由后操作的

$ ssh root@192.168.1.1

挂载U盘

安装必须的包，路由能联网就直接执行安装

$ opkg update
$ opkg install kmod-usb-storage block-mount kmod-fs-ext4

没网的去官方库里下载相应的包，使用scp命令复制到路由安装即可
官方库:https://downloads.openwrt.org/chaos_calmer/15.05/

当然也可以自己编译固件时包含这三个包，刷好了继续下面的操作

将U盘插上，执行block info命令，看下U盘是否已被识别,会有sda1,和sda2，分别对应ext4分区和swap分区

下面操作默认sda1 为ext4分区，sda2 为 swap 分区

转移系统到U盘

$ mount /dev/sda1 /mnt
$ mkdir /tmp/root
$ mount -o bind / /tmp/root
$ cp /tmp/root/* /mnt -a
$ umount /tmp/root

开机自动挂载

1、修改/etc/config/fstab ，加入下面内容

config mount
        option target   /overlay
        option device   /dev/sda1
        option fstype   ext4
        option options  rw,sync
        option enabled  1
        option enabled_fsck 0

2、执行如下命令生效

$ /etc/init.d/fstab enable

3、设置 swap 分区,修改 fstab 文件

config swap
        option device   /dev/sda2
        option enabled  1

4、激活分区

$ mkswap /dev/sda2

5、重启路由

$ reboot

重启后

再看fstab是下面这个样子的

查看挂载信息

$ df -h

看到/overlay 空间已经是5G了

接下来就愉快的安装软件吧,再也不用担心存储空间的问题了！
