Windows安装配置OpenCV

发表于 2016-02-25 | 更新于 2016-05-06 | 分类于 OpenCV

前言

一直以来都是在Visual Studio中使用OpenCV，但是VS真的太大了，于是想直接就在Qt里使用OpenCV。
遇到一个问题，就是Qt一直找不到头文件，而且不只是在OpenCV上，对于别的第三方库也是。
折腾一晚上，终于发现问题，就是Qt的bin目录得加到系统的环境变量PATH里。
对于在Visual Studio中使用Qt是不需要把Qt的bin目录加到环境变量的，但是如果想用Qt做工程管理，就需要配置环境变量。

总结一下，把OpenCV在VS2015和Qt Creator中的部署一起写出来。

各软件版本如下：
OpenCV 3.1
Visual Studio 2015
Qt 5.6
Qt Creator 3.6.1 Based on Qt 5.6.0 (MSVC 2013, 32 bit)

安装OpenCV

OpenCV:http://opencv.org/
当前版本（3.10）

解压opencv310到OpenCV目录。
比如我的C:\OpenCV\opencv310

添加两个系统变量
OPENCV_DIR = C:\OpenCV\opencv310
OPENCV_VER = 310

Windows10 在PATH中新建
%OPENCV_DIR%\build\x64\vc14\bin

Windows10 管理环境变量非常方便，一行一项。
非Windows10的，与前一项以分号隔开。

安装完成

提示：

  • 1、如果想在OpenCV 3.x系列之间切换，只需要将系统变量的OPENCV_DIR和OPENCV_VER改为相应的值就可以了。
  • 2、OPENCV_VER并不是必须的，如果不定义，请将属性表中所有的$(OPENCV_VER)替换成OpenCV版本号，比如3.1版，就是310.
