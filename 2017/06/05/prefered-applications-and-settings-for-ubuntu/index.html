SwitchyOmega

安装
安装方式一：Google Web Store （能上Google的前提下推荐使用）

安装方式二：Github最新离线安装包 （不能上Google时使用），下载.crx后缀的文件，直接拖到chrome的插件管理页即可安装

配置
配置文件：GFWList.bak

在“导入/导出”位置，选择“从备份文件恢复”，选择下载的GFWList.bak即可导入，或 选“在线恢复”，直接粘贴配置文件的地址。

选择自动切换后更新规则即可。

GFWList项目地址：https://github.com/gfwlist/gfwlist

规则列表地址： https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt
